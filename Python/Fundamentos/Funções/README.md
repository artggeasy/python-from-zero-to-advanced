# 🧪 Exercícios em Python

Este repositório contém exercícios práticos em Python com foco no desenvolvimento da lógica de programação e preparação para back-end.

Os exercícios vão do básico até pequenos desafios que simulam situações reais.

---

## 📌 Objetivo

- Praticar lógica de programação
- Melhorar a escrita de funções
- Entender fluxo de decisão (if/else)
- Criar base sólida para desenvolvimento back-end

---

## 🧪 Exercícios

### 🔹 Exercício 1

Crie uma função que:
- recebe um número
- retorna o dobro

### 🔹 Exercício 2

Crie uma função que:
- recebe um nome
- retorna: "Olá, nome"

### 🔹 Exercício 3

Crie uma função que:
- recebe usuário e senha
- se for "admin" e "123" → retorna "Login OK"
- senão → "Erro"
