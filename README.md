# 🚀 Python: python-from-zero-to-advanced (Back-End)

Este repositório documenta minha jornada de aprendizado em Python com foco em desenvolvimento back-end.

A ideia é evoluir do básico até conceitos avançados, aplicando na prática com exercícios, projetos e estudos organizados.

---

## 📌 Objetivo

- Aprender Python de forma estruturada
- Evoluir para desenvolvimento back-end profissional
- Criar um portfólio sólido no GitHub
- Praticar lógica de programação e resolução de problemas
- Trabalhar com tecnologias reais do mercado

---

## 🧠 Conteúdo

Este repositório está dividido por níveis:

### 🟢 Nível 1 — Fundamentos
- Variáveis e tipos de dados
- Operadores
- Estruturas condicionais (if/else)
- Loops (for, while)
- Funções

### 🟡 Nível 2 — Intermediário
- Listas, tuplas e dicionários
- Manipulação de strings
- Funções avançadas
- Tratamento de erros (try/except)
- Módulos e pacotes

### 🔵 Nível 3 — Avançado
- Programação orientada a objetos (POO)
- Decorators
- Lambda e funções funcionais
- Manipulação de arquivos
- Introdução a testes

### 🔴 Nível 4 — Back-End
- APIs com Flask / Django
- Rotas e requisições HTTP
- Integração com banco de dados (SQLite/PostgreSQL)
- Autenticação
- Estrutura de projetos reais

---

## 🛠️ Tecnologias

- Python 3
- Flask (em breve)
- Django (em breve)
- SQLite / PostgreSQL
- Git & GitHub
